# Reorder

Reorder is how you restock something you already carry. You find the product, set how many you want in each size, send the order to the supplier, and receive it when it lands. It is a separate order from the original buy, so your numbers stay clean.

## Quick path

<ol class="steps">
  <li><strong>Open Reorder</strong> from the sidebar.</li>
  <li><strong>Find the product</strong> — search for one you carry, or click <strong>Run</strong> on a saved list.</li>
  <li><strong>Set how many, by size</strong> — type the quantities. They save on their own.</li>
  <li><strong>Send to the supplier</strong> on the PO tab (or use <strong>Save &amp; Receive</strong> to skip it).</li>
  <li><strong>Check it in</strong> on the Receive tab when the box arrives.</li>
</ol>

<div class="callout callout-tip">
<div class="callout-title">Where "what should I reorder?" lives</div>
The reorder screen is for placing the restock. To decide <em>what</em> to restock — what's selling, what's running low — use the <a href="/planning/trading/">Trading dashboard</a>. It surfaces reorder candidates from sell-through and weeks-of-supply, and links straight into a reorder.
</div>

## Two ways to start

Open <strong>Reorder</strong> from the sidebar. The page is a search, split into two panels:

- <strong>Products</strong> (left) — search your Shopify catalog. Type at least 2 letters. Each result shows a thumbnail, brand, colour, stock count, and dots for how complete the product data is. Click a product to start a reorder.
- <strong>Saved lists</strong> (right) — your saved restock plans. <strong>Recent</strong> shows the last few; <strong>Browse by brand</strong> groups the rest. Click <strong>Run</strong> on any list to start a reorder pre-filled with those products and sizes.

A reorder is locked to one brand. If you try to add a product from a different brand, the app says so and blocks it. Run a second reorder for the second brand.

## The Pick / PO / Receive tabs

Once you start, you're in the reorder workspace. Three tabs across the top track where you are:

- <strong>01 Pick</strong> — choose products and quantities
- <strong>02 PO</strong> — review and send to the supplier
- <strong>03 Receive</strong> — check in what arrives

## 01 Pick — set quantities

Each product shows a row of size cells. Type how many you want in each size.

- <strong>Quantities save on their own</strong> about a second after you stop typing. There is no Save button — watch for the small <em>Saving…</em> then <em>Saved</em> note.
- Each size cell shows a <strong>stock badge</strong> so you know what's already on the floor:
  - <strong>0</strong> — out
  - <strong>1–3</strong> — low
  - <strong>4+</strong> — in stock
  - <strong>—</strong> — not in Shopify yet (stock unknown)
- If you started from a saved list, the cell also shows the plan snapshot, e.g. <em>plan 20 · have 15</em>.
- Trousers and other two-axis sizes (waist × inseam) lay out as a grid, not a single row.

<div class="callout callout-tip">
<div class="callout-title">Earmark for a waiting customer</div>
If a customer is waiting on a size, click <strong>add earmark</strong> on that cell, type their name (or pick a Shopify customer). The earmark guarantees that size is on the order — the quantity bumps up to cover it. Earmarks show on the Receive tab so you know who to call when it lands.
</div>

Other things on the Pick tab:

- <strong>Cancel after</strong> — an optional date the supplier shouldn't ship past.
- <strong>Order discount</strong> — a percent off the whole order. (Per-line discounts grey out when this is set; they never stack.)
- <strong>Add another product</strong> — search at the bottom to add more products from the same brand.
- <strong>Save as plan</strong> — save these products and sizes as a reusable restock list. Next season, click <strong>Run</strong> and it's all there.

When you're done, use the footer:

- <strong>Go to PO</strong> — the normal path (send to the supplier first).
- <strong>Save &amp; Receive</strong> — skip the PO and go straight to receiving. Use this when you're not sending a formal PO (e.g. you already placed the order by phone). Only available before a PO has been sent.

## 02 PO — send to the supplier

The PO tab shows the purchase order for the supplier: styles, sizes, quantities, and the supplier's contact. From here you can:

- <strong>Email brand contact</strong> — opens an email to the supplier, pre-filled.
- <strong>Generate PO link</strong> / <strong>Download PO</strong> — a clean copy to send or print.
- <strong>Mark PO as Sent</strong> — records that it's out. This moves you to the Receive tab.

<div class="callout callout-info">
<div class="callout-title">If the supplier confirms changes</div>
Some suppliers confirm availability per size before shipping. When they do, you'll see two ways to accept: <strong>Accept as quoted</strong> (take what they offered) or <strong>Accept reduced quantities</strong> (they couldn't fill everything). Whichever you pick pre-fills the Receive tab, so you're not counting against quantities the supplier already said they can't ship.
</div>

## 03 Receive — check it in

When the box arrives, open the Receive tab. It uses the same size grid as Pick. Each cell shows what's still outstanding and a quantity box pre-filled for you:

- the supplier's confirmed quantity, if they responded;
- 0 if they marked that size unavailable;
- the full remaining amount if there was no supplier response.

Adjust any cell for a partial shipment, then click <strong>Confirm receive</strong>. Sizes turn green when fully received. You can come back and receive again for later boxes.

## If it goes wrong

- <strong>"That product belongs to a different brand"</strong> — a reorder is one brand only. Finish this one, then start another for the other brand.
- <strong>Quantities didn't save</strong> — they save a second after you stop typing. If you see <em>Save failed</em>, check your connection and re-type the cell.
- <strong>Stuck status, or anything else</strong> — see <a href="/runbook/">If something breaks</a>.
